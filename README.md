<h1> Sales data </h1>

<b>3 Step project</b>

1- Data Pipeline Automation – Built an automated data ingestion and transformation pipeline using Python and SQL on AWS S3 and EC2.

2-Snowflake Data Warehouse – Designed and implemented a Snowflake data warehouse with structured schema and efficient data loading using SQL scripts.

3-Power BI Integration – Created interactive dashboards in Power BI connected to SQL databases to visualize business KPIs.


<b>Goals</b>

Load and clean sales data

Create anew KPI column : profit margin ($)

Visualise insights with matplotlib and seaborn

Simulate ETL and data warehousing using SQLite
